## Getting Started

1. Clone the repo:

```
git clone https://github.com/IgorSprovieri/weekly-web
```

2. Open in the explorer the project

```
Open the index.html file with some browser
```

3. You can create your own user to test

```
Click on "Criar Conta" button
```
